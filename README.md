# Hand Tracking Drawing Application

This repository contains a Python application that uses OpenCV, Mediapipe, and NumPy for hand tracking and drawing on the screen.
The program detects hand gestures and allows the user to draw by controlling a virtual pen with their fingers.

## Features

- Hand tracking using Mediapipe
- Draw on the screen using the index finger only when straightened
- Clear the drawing by touching the tips of your pinky and thumb
- Real-time display of hand landmarks

## Requirements

- Python 3.x
- OpenCV
- Mediapipe
- NumPy
